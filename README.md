# Banking App Repo
This is a banking app project for the Code2College Elite 102 course. The program has been completed with functions such as account viewing and creation, deposits and withdrawals, balance checking, and a menu CLI.
## Instructions
This is best done as a codespace with access to a database visualizer extension such as DB Viewer (highly recommended). I have used it in my program to more easily visualize my database in the tutorial.db file. Once you have a codespace set up, run the program and create, view, deposit, and withdraw from your account. This program mainly works with one account at a time.
## Features:
- Account Viewing
- Account Creation
- Withdrawals
- Deposits
- Checking your Balance
